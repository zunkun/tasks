![calendar](./images/calendar.png)

## 部署
1. 本工具使用 [zwork](https://github.com/zunkun/zwork/server)作为服务，请优先部署
2. `cd tasks && npm install` 安装依赖
3. `npm run dev`开发

你也可以直接运行`zwork` 访问 `http://localhost:5500/tasks`
